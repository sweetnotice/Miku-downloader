1. 把链接一行一个复制进 批量下载队列 里 再打开 下载器就可以批量下载

1. 使用onetab插件，加入你想要下载的番剧<img src="README.assets/image-20220903125927695.png" alt="image-20220903125927695" style="zoom: 80%;" />
